# 15 Nouvelles Questions pour le Test d'Âge Biologique

## Catégorie : Santé Cardiovasculaire (3 questions)

### Q1: Tension artérielle
- **Question:** Connaissez-vous votre tension artérielle ?
- **Type:** Select
- **Options:**
  - Normale (< 120/80) → -2 ans
  - Pré-hypertension (120-139/80-89) → +1 an
  - Hypertension légère (140-159/90-99) → +3 ans
  - Hypertension modérée/sévère (≥160/100) → +5 ans
  - Je ne sais pas → +1 an

### Q2: Fréquence cardiaque au repos
- **Question:** Votre fréquence cardiaque au repos (battements/min)
- **Type:** Range
- **Valeurs:** 40-100 bpm
- **Impact:**
  - < 60 bpm → -2 ans (excellent)
  - 60-70 bpm → -1 an (très bon)
  - 71-80 bpm → 0 (normal)
  - 81-90 bpm → +2 ans
  - > 90 bpm → +4 ans

### Q3: Antécédents familiaux cardiovasculaires
- **Question:** Antécédents de maladies cardiaques dans votre famille proche ?
- **Type:** Radio
- **Options:**
  - Non → 0
  - Oui, après 70 ans → +1 an
  - Oui, avant 60 ans → +3 ans
